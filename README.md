# sift FreeBSD Port

[sift - grep on steroids](https://sift-tool.org/) - fast and powerful alternative to grep.

## Usage
```
fetch --no-verify-peer \
  http://github.com/wakeful/ports-sift/archive/master.zip \
  ; tar -xvf master.zip \
  ; rm master.zip \
  ; cd ports-sift-master \
  ; make clean install
```
